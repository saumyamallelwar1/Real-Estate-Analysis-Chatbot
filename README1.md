 # Real Estate Chatbot - Frontend
 React-based chat interface for real estate analysis.
 ## Setup
 1. Install dependencies:
 ```bash
 npm install
 ```
 2. Start development server:
 ```bash
 npm start
 ```
 3. Build for production:
 ```bash
 npm run build
 ```
 ## Features- Chat-based UI- Real-time data visualization- File upload- Responsive design
